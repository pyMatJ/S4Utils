Welcome to S4Utils's documentation!
===================================


What is it?
-----------

S4Utils is a collection of python utility functions to ease the use of the `S4 software <http://web.stanford.edu/group/fan/S4/>`_ [#f1]_ 

.. rubric:: References

.. [#f1] Liu, V. and Fan, S., *Computer Physics Communications* **183**, 2233-2244 (2012) `10.1016/j.cpc.2012.04.026 <https://doi.org/10.1016/j.cpc.2012.04.026>`_
